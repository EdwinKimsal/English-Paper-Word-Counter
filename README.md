# English Paper Word Counter

This Python script analyzes a `.docx` file and finds the most frequently used words and phrases.

## Features
- Counts **single words** (stop words removed)
- Counts **two-word phrases (bigrams)**
- Counts **three-word phrases (trigrams)**
- Case-insensitive
- Removes punctuation

## Requirements
- Python 3.x
- `python-docx` library

Install the required library:
```bash
pip install python-docx
